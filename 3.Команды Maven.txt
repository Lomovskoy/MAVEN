1. mvn archetype:generate - генерирует новый проект maven
2. Будет ждать ввода - ввести слово для фильтрации найденых архетипов
	: quickstart нажать энтер, покажутся только отфильтрованные рахетипы,
3. Повторить ввод, будет выбран архетип quickstart по умолчанию.
4. Надо выбрать версию архетипа - лучше последнюю. Ввод без параметров.
5. Попросит ввести groupId: допустим mygroupmavenproject
6. Попросит ввести artifactId: допустим exemple
7. Версия проекта по умолчанию
8. Попросит ввести package: допустим mavenproject
9. Подтвердить Y
